# QuoteCompare

AI-powered freight and vendor quote comparison tool.

## What It Does

- Upload multiple vendor quotes (PDF, Excel, images)
- AI extracts pricing automatically  
- Side-by-side comparison table
- Instant recommendations

## Tech Stack

- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express + OpenAI API
- **Deployment**: Vercel (frontend) + Railway (backend)

## Run Locally

```bash
# Install
npm install

# Start dev server
npm run dev
```

Visit: http://localhost:5173
